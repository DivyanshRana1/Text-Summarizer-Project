Text Summarizer Using TextRank Algorithm

Objective:
Generate a summary of input text using Natural Language Processing and Graph-based Ranking.

Libraries Used:
- NLTK
- NetworkX
- NumPy

Algorithm:
- Sentence Tokenization and Preprocessing
- Similarity Matrix Computation
- PageRank for Sentence Importance
- Top N Sentences as Summary

How to Run:
1. Install dependencies:
   pip install -r requirements.txt
2. Download NLTK data:
   import nltk
   nltk.download('punkt')
   nltk.download('stopwords')
3. Run script:
   python text_summarizer.py
